# Election Voting System

- This project is a prototype for the real time election voting system. With some modifications it can be used for college level elections

## Features of this application

    > Shows charts to compare candidates performance
    > Validates ids of voters accordingly
    > Provides a choice to take care of draw

# **Note**

    > If you wish to make any changes or use the application, make sure to keep a csv file with ids of people eligible to vote

# Future work
